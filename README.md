## איך להשתמש

### למרצה:
1. **יצירת GPTs:** העתק את תוכן כל קובץ מ-`gpt-instructions/` ל-Custom Instructions של GPT חדש ב-ChatGPT
2. **עדכון קישורים:** ב-`lesson-scenarios.html`, עדכן את אובייקט `agentUrls` עם הלינקים ל-GPTs
3. **בדיקה:** השתמש ב-`testing-checklist.md` לבדוק כל GPT לפני השיעור
4. **הפעלה:** שתף את הלינק ל-GitHub Pages או את קובץ ה-HTML עם הסטודנטים

### לסטודנטים:
1. פתח את `lesson-scenarios.html` בדפדפן
2. בחר את קבוצת התרגול שלך (1-4)
3. לחץ על "התחל שיחה עם הסוכן" — תקבל 15 דקות לתחקור
4. שאל שאלות חכמות כדי לחלץ מידע למסמך האפיון
5. מלא את מסמך האפיון (חלקים א'-ד')
6. **אחרי סיום** — השתמש בקוד PIN לגשת לחומרי עזר

## מסמך האפיון — מה הסטודנטים ממלאים

| חלק | תוכן | מה צריכים לחלץ מהתחקור |

| ד' | הפיצוח | קונספט, ארגון תוכן, ניווט, חווית לומד |
